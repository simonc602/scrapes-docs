---
title: "First verification"
description: "Check that the hosted Team OS deploy is ready."
---

Run these checks after deployment.

1. Open `/v1/health` and confirm `ok: true`.
2. Confirm the health response says `backend: postgres`.
3. Sign in from a local client.
4. Run `npm run team -- whoami`.
5. Run `npm run team -- clients`.
6. Open Command Centre.
7. Confirm the Team tab can load memory status.

If a check fails, start with
[Team OS troubleshooting](/docs/admin/team-os-troubleshooting).
