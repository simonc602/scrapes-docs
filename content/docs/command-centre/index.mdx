Command Centre is the local interface for Agentic OS. Start in **Feed** to
create a Goal, follow its tasks, and open the files and plans connected to the
work.

## Main areas

| Area | Use it to |
| --- | --- |
| Feed | Create Goals, continue chats, and review work. |
| Scheduled | View and manage scheduled work. |
| Team | Work with Team OS. This area appears when Team OS is available. |
| Skills | Find and manage the skills available to Agentic OS. |
| Docs | Browse workspace context, documentation, and project files, including available Team and client sync controls. |
| Settings | Change local Command Centre settings. |

## Start here

1. Open [Feed](/docs/command-centre/feed).
2. Select a workspace and create a Goal.
3. Choose the right [task permission mode](/docs/command-centre/task-permission-modes)
   before you send the first message.

If you use a hosted team, connect Command Centre to Team OS before starting
work that needs shared clients, skills, or memory. See
[Connect from Command Centre](/docs/team-os/command-centre-login).
