---
title: "Published memory"
description: "Edit or delete published shared memory."
---

Owners and admins can manage published memory in Command Centre.

1. Open the Team tab.
2. Open Memories.
3. Use search or the status, scope, and client filters.
4. Open a published memory.
5. Edit it, or delete it from recall.

Editing a published memory updates its indexed content.

Deleting a published memory archives the source and removes its search chunks.
The audit trail still records the delete action.
