---
title: "Team OS"
description: "Team OS connects local Agentic OS workspaces to shared Companies and Teams."
---

Team OS connects local Agentic OS workspaces to a hosted server. One Company can
contain multiple Teams, with separate access and resources in each Team.

The hosted server is the source of truth for users, Company access, Team roles,
clients, grants, shared memory, and file access. A local workspace signs in to the
server and receives only the context allowed for the selected Team.

Start here before using the hosted setup and client connection guides.
