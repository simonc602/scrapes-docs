---
title: "Owner password reset"
description: "Create a hosted owner password reset link."
---

Use owner reset when a hosted owner needs a password reset link.

Run this where the hosted Team OS environment is available. If you are already
inside the `memory-api` container, run:

```powershell
npm run team:owner-reset
```

From the Docker host, first find the container name:

```powershell
docker compose ps
```

Then run the command inside that container:

```powershell
docker exec -it <memory-api-container-name> npm run team:owner-reset
```

If the command cannot detect the public URL, pass it explicitly:

```powershell
npm run team:owner-reset -- --base-url https://team.example.com
```

From `command-centre/`, you can also choose the team and owner:

```powershell
npm run team:owner-reset -- --team demo --email owner@example.test
```

The command prints a reset URL. Treat it like a secret because it can change the
owner password until it expires.
