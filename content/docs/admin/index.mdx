Use this section for owner and administrator tasks.

Start here when you need to manage users, clients, grants, skill access, memory
review, owner recovery, health checks, or manual verification.

For most admin work, use the Command Centre Team tab or the interactive
terminal menu. Direct commands are still documented for reference.
