Use this section to manage Team OS access and operations.

One Team OS server represents one company. Company roles control access across
that company. Direct Team roles and grants control access inside each Team.

Use Command Centre for normal admin work. Use maintenance commands only for
bootstrap, recovery, automation, or troubleshooting.
