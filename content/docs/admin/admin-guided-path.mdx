---
title: "Admin guided path"
description: "Use the Command Centre Team tab or the interactive terminal menu."
---

You can run admin actions in either place:

* Command Centre Team tab for the visual UI.
* Interactive terminal menu for a guided terminal flow.

To open the terminal menu, run this from the Agentic OS workspace:

```powershell
cd command-centre
npm run team
```

![Team OS interactive terminal menu](/assets/team-os-interactive-menu.png)

The menu lets you choose actions for team status, members, clients, secrets,
skills, context and sync, and memory.

You do not need to remember every command for normal admin work.
