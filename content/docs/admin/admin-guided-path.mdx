---
title: "Admin guided path"
description: "Use Command Centre or the interactive terminal menu."
---

Use the surface that matches the task:

* **Team > Teams** for Company Owner and Company Admin work.
* The selected Team's sections for members, clients, memory, secrets, and
  skills.
* Interactive terminal menu for a guided terminal flow.

To open the terminal menu, run this from the Agentic OS workspace:

```powershell
cd command-centre
npm run team
```

The menu covers the selected Team. Company-wide access requests, Company Admin
grants, Team lifecycle, and ownership transfer are clearer in Command Centre.

You do not need to remember every command for normal admin work.
