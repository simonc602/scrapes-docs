---
title: "Run automated checks"
description: "Run automated Team OS checks after manual verification."
---

Run:

```powershell
npm run test:identity
npm run test:isolation
npm run test:memory
npm run build
```

The identity checks cover Company roles, Team access requests, ownership,
invites, Full access, and Member grants. The manual flow is ready when all
checks pass and the expected results match the local run.
