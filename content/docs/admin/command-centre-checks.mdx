---
title: "Command Centre checks"
description: "Check admin controls in Command Centre."
---

Open **Team** after signing in.

Company Owners should see **Teams** with these tabs:

* **All Teams**;
* **Requests**;
* **Company Admins**.

Company Admins should see:

* **All Teams**;
* **My Access**.

After selecting a Team, users with Full access should see:

* members;
* clients;
* client grants;
* skill grants;
* secrets;
* memory status;
* memory review;
* published memory actions;
* sync state.

Members should not see admin-only management controls. Their visible Team
sections include **Dashboard**, **Clients**, **Secrets**, and **Skills**. The
data and actions inside those sections still follow their grants.

The client list must come from the Team OS backend in hosted mode. Full access
users receive every active client with implicit write access. Members receive
only explicitly granted clients. The UI must not fall back to every client found
in the local workspace folder when Team OS is connected.
