---
title: "Health checks"
description: "Check the hosted Team OS API."
---

Check the hosted API:

```powershell
Invoke-RestMethod -Uri "https://team.example.com/v1/health" -Method Get
```

Expected response:

```json
{
  "ok": true,
  "backend": "postgres",
  "embedder": {
    "model": "bge-m3",
    "dim": 1024
  }
}
```

If hosted mode is expected, `backend` should be `postgres`.

If it says `pglite`, the API is not connected to hosted Postgres.
