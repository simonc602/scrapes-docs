---
title: "Review pending memory"
description: "Review staged Team OS memory before it enters recall."
---

Users with Full access review pending memories in Command Centre. This includes
the Company Owner, a Team Owner or Team Admin, and a Company Admin with Full
access to the selected Team.

1. Sign in to Team OS.
2. Select the Team.
3. Open **Team > Memories**.
4. Review pending items.
5. Publish or discard each item.

The review UI uses `/v1/memory/memories`. It shows pending review items and
published team or client memory.

Team Members do not get the review controls.

When a reviewer publishes an item, Team OS creates final memory and links it
back to the original capture.

When a reviewer discards it, the item stays out of recall.

See [Roles and permissions](/docs/team-os/roles-and-permissions) for every
source of Full access.
