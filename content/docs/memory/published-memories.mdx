---
title: "Published memories"
description: "Manage published team and client memory."
---

Users with Full access can manage published Team and client memories. This
includes the Company Owner, a Team Owner or Team Admin, and a Company Admin
with Full access to the selected Team.

1. Sign in to Team OS.
2. Select the Team.
3. Open **Team > Memories**.
4. Select a published memory.
5. Edit the memory, or delete it from recall.

Editing a published memory re-indexes its content so search uses the latest
text.

Deleting a published memory archives the memory source and removes its search
chunks. The audit trail still keeps the delete event.

Team Members can search Team memory and memory for clients they can access.
They do not get the edit or delete controls for published memories.

See [Roles and permissions](/docs/team-os/roles-and-permissions) for every
source of Full access.
