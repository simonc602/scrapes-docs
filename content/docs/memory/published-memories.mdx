---
title: "Published memories"
description: "Manage published team and client memory."
---

Owners and admins can manage published team and client memories.

1. Sign in to Team OS.
2. Open the Team tab.
3. Open Memories.
4. Select a published memory.
5. Edit the memory, or delete it from recall.

Editing a published memory re-indexes its content so search uses the latest
text.

Deleting a published memory archives the memory source and removes its search
chunks. The audit trail still keeps the delete event.

Members can search memory that their grants allow. They do not get the edit or
delete controls for published memories.
