# Agentic OS Documentation

![Agentic OS logo](assets/agentic-os-logo.png)

Agentic OS helps people run repeatable AI work across local workspaces, clients,
shared memory, skills, scheduled jobs, and Team OS.

Start with [Get started](get-started/README.md), then read
[Team OS](team-os/README.md) when you are ready to connect a local client to a
hosted server.

Use these docs in this order:

1. Install Agentic OS locally.
2. Learn the core terms and folder structure.
3. Add clients when you need separate workspaces.
4. Use Team OS when a hosted server must control team access.
