# Agentic OS

Agentic OS is a workspace system for AI-assisted work. It gives Claude Code and
Codex shared instructions, skills, memory, project folders, scheduled jobs, and
a local UI called Command Centre.

Use this section to understand the local system before you add more clients or
connect to Team OS.
