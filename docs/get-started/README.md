# Scrapes Docs

Scrapes Docs covers Agentic OS and Team OS: local AI workspaces, shared memory,
skills, Command Centre, hosted teams, client access, and safe file sync.

Start with local setup if you are using Agentic OS for the first time. Start
with Team OS deploy when you need a hosted server for a team.
