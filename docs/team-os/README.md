# Team OS

Team OS connects local Agentic OS clients to a hosted server.

The hosted server is the source of truth for users, teams, clients, roles,
grants, shared memory, and permission-scoped file access. A local client signs
in to the server and receives only the context it is allowed to use.

Start here before using the hosted setup and client connection guides.
